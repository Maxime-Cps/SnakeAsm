# SnakeAsm

## First Year Project

### Working Time

30h of work in school and at home

### Features

- Snake's Movement
- Head Directions
- Candies and Obstacles Display
- Handled End-Game Conditions
- Handled Scoring
- Handled Snake's Tail Movement and Growth

### How To Play

- Download the .jar file [Mars4_5.jar](Mars4_5.jar)
- Execute Mars
- Open the [snake.asm](snake.asm) file in Mars
- Go to Tools > BitMap Display
- Set the Unit Width and Height to **16**
- Set the Display Width and Height to **256**
- Go to TOO
